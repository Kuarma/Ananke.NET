# Ananke.NET
A self-hosted network management tool for controlling your own PCs on your local network.

## Installation
1) Clone the repo:
```bash
git clone https://github.com/Kuarma/Ananke.NET
cd Ananke.NET
```
2) Restore packages:
```bash
dotnet restore
```
3) Build project:
```bash
dotnet build
```
4) Run the project:
```bash
dotnet run
```
